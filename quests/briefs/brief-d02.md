# 📋 Quest Brief · Code & Engineering

**Сфера:** T · **Перший практичний крок**

## 🎯 Мета
Зібрати **одну видиму річ** — кнопку, форму або маленький скрипт.

## 🕵️ Кроки
1. Обери **один** стек (HTML+JS або Python CLI — не обидва).
2. Список з **3 функцій** max для v0.
3. Закоміть у GitHub — навіть якщо «некрасиво».

## ⚠️ Не роби так
Не проектуй архітектуру мікросервісів для першої кнопки.
