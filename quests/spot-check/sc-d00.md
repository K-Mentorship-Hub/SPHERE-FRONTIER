# 🔍 Spot Check · 7 Days Challenge

**Сфера:** Bridge · **Напрям:** 0

## 🎯 Мета
Перевірити, чи тест-драйв справді різноманітний — чи це три однакові задачі в новій упаковці.

## 📋 Сценарій
Менті пише: «За 7 днів спробувала SQL, Python і Figma — обрала напрям». Але всі три міні-задачі були з **Technology**, жодної з Science чи Entrepreneurship.

## 🕵️ Твоє завдання
1. Чи це справжній тест-драйв по SET?
2. Що саме вона **не** перевірила?
3. Як переписати план на 7 днів, щоб компас працював?

## ⚠️ Пастка
«Три інструменти» ≠ «три сфери». Різноманітність — по **питаннях**, не по софту.

<summary>✅ Перевір себе (лише після своєї відповіді)</summary>

**Ні**, це не повний test-drive: усі три — T. Вона не перевірила S (дослідницьке питання / джерело) і E (користувач / валідація).

**Краще:** День 1 — одне S-міні (глосарій + одне джерело). День 2 — E (5 питань людині про проблему). День 3 — T (маленький SQL або кнопка). Потім рефлексія «що заходить».
