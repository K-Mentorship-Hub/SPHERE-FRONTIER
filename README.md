# K Mentorship Hub

Beta hub for mentorship, learning paths, and future community operations.

## Current Positioning

This repo is intentionally lightweight.

Right now the goal is not to run a full Discord ecosystem. The goal is to make the concept legible:

- what the hub is
- who it is for
- what the first intake should look like
- what will come later

## Recommended Near-Term Scope

- one public beta page
- one intake form
- one Notion CRM destination
- no heavy community operations yet

## Future Later

- Discord community
- program tracks
- automations and role routing
- talent pool only if there is real demand
